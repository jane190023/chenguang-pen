# chenguang-pen 晨光智能笔

> 基于 AI 的纸笔学习智能陪伴系统，让普通纸笔也能记录学习数据

## 项目简介
晨光智能笔是一款面向学生群体的纸笔学习智能陪伴系统。通过轻量化的分体式感应配件，在不改造中性笔本身的前提下，实现书写行为数据的自动采集与 AI 分析，帮助学习者直观了解学习状态、培养良好学习习惯。
针对当前学生普遍存在的专注力不足、学习节奏不稳定、缺乏持续反馈等痛点，本方案将 AI 技术与传统书写场景深度结合，通过实时记录书写时长、书写量、停顿频率等数据，量化分析专注状态，提供即时休息提醒，并形成长期个性化学习档案，让每一次动笔都有迹可循。

## 核心功能
### 专注力可视化引擎
- 基于书写时长、停顿间隔、答题节奏等多维度数据
- 实时量化分析专注力波动状态
- 生成专注度趋势图表，直观呈现学习状态变化

### 即时反馈提醒
- 自动统计连续书写时长
- 智能提醒劳逸结合、适时休息
- 帮助建立科学的学习节奏

### 个性化学习档案
- 长期沉淀学习行为与书写数据
- 动态生成周 / 月学习习惯报告
- 可视化展示学习积累，提供持续正向激励

## 技术架构

整体采用四层架构设计，自下而上分别为：

| 层级 | 名称 | 说明 |
|------|------|------|
| 第一层 | 感知采集层 | 指环传感器、笔托配件、书写数据采集 |
| 第二层 | 数据处理层 | 数据清洗、特征提取、统计计算 |
| 第三层 | AI 能力层 | 专注度研判、习惯分析、反馈生成 |
| 第四层 | 应用交互层 | 移动端 / 网页端数据展示与交互 |

## 产品特色

### 分体式设计，考场可用
- 普通笔杆无任何电子元件，可正常带入考场
- 居家学习时搭配智能配件即可记录数据
- 兼顾考试需求与日常学习数据化

### 全品牌兼容
- 通用夹持笔托设计，适配市面上绝大多数中性笔
- 无需购买专用笔芯，耗材成本低
- 沿用日常书写习惯，学习零门槛

### 轻量化方案
- 无需对笔体进行复杂硬件改造
- 以软件算法为核心，硬件成本可控
- 依托晨光现有生产与渠道资源，落地可行性高

## 调研数据支撑
基于 50 份学生群体有效问卷调研：
- **94%** 的学生无法准确掌握自己的学习时长，仅 6% 会主动计时
- **94%** 的受访者对自动统计学习数据的工具持期待或中立态度
- **68%** 的消费者因定价偏高放弃一体式智能笔，60% 因考场受限放弃
- **56%** 的学生明确表示愿意体验分体式智能配件方案

数据表明，分体式、兼容通用中性笔的智能书写方案具有明确的市场需求。
